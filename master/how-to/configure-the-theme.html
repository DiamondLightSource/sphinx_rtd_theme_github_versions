Configure the theme¶

The following options can be defined in your project’s conf.py file, using the html_theme_options configuration option.

For example:

html_theme_options = {
    'versions_branches': 'main master',
    'github_org': 'dls-controls',
}

Theme Options¶

versions_branches¶

The branches that should appear top of the list of versions before all the tags. Should be supplied as a space separated string.

Type

str

Default

"main master"

github_org¶

The github organization that the repo is hosted under

Type

str

Default

"dls-controls"

All other sphinx_rtd_theme options like style_nav_header_background are also supported.
